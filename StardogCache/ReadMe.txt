Installation: 

- This program requires Java 8 to build and run, please update your Java if necessary
- This program requires Stardog-4RC-3, please install Stardog. Just google it and follow its excellent manual
- Apache ant is also required to compile this code, install it if necessary.
- unzip LUBM45.nt.zip into LUBM45.nt, this is the data source for streamign data emulation. It's important to unzip this file into the same directory.
After you have everything, open up your terminal, go to the directory that contains this program, enter "ant", it will run.
